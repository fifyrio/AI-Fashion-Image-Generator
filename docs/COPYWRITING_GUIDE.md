# 爆款文案生成功能使用指南

## 功能介绍

AI 爆款文案生成功能可以帮助你分析现有的优秀文案，并基于其爆款要素生成 3 个类似风格的新文案。该功能已集成到主应用的"生成类似文案"标签页中。

## 爆款公式

系统内置了两种经典爆款公式：

### 公式 1（简洁型）

**结构**: 钩子开场 → 产品细节拆解 → 搭配建议 → 产品优势 → 使用场景/行动号召

**示例**:
```
不会搭的男生看这里！一件搞定秋冬造型
麻花针织纹理超有细节感，半拉链设计比普通毛衣特别，深色系超好搭配外套。
实测洗后不变形，男生衣橱必备基础款！
#男生基础款 #好搭单品 #秋冬必备 #男士针织衫 #懒人穿搭
```

**特点**:
- 直接切入，简洁有力
- 突出产品细节和功能优势
- 适合男装、基础款商品
- 强调实用性和性价比

### 公式 2（故事型）

**结构**: 开场白（情绪钩子）→ 使用场景（真实故事）→ 产品细节拆解 → 痛点解决 → 行动号召

**示例**:
```
姐妹们，这套真的太绝了！
今年过年就穿这套出门拜年，亲戚朋友都夸好看。
上衣是新中式盘扣设计，特别显气质，加棉的很暖和。
裙子高腰的，遮小肚子还显腿长，简直小个子女生的福音，喜欢的姐妹千万别错过！
#甜美穿搭 #温柔穿搭 #韩系穿搭
```

**特点**:
- 带入真实使用场景
- 强调情感共鸣和真实体验
- 适合女装、时尚单品
- 注重痛点解决和情绪价值

## 使用方法

### 1. 在 Web 界面使用

1. 启动应用: `npm run dev`
2. 打开浏览器访问 `http://localhost:3000`
3. 点击"生成类似文案"标签页
4. 输入原始文案
5. 选择目标群体（男生/女生）
6. 点击"生成类似文案"按钮
7. 查看分析结果和生成的 3 个新文案
8. 点击"复制文案"按钮复制你需要的文案

### 2. 使用 API 测试脚本

```bash
# 确保开发服务器正在运行
npm run dev

# 在另一个终端运行测试脚本
npx tsx scripts/test-copywriting.ts
```

### 3. 直接调用 API

**端点**: `POST /api/generate-similar-copywriting`

**请求体**:
```json
{
  "originalCopy": "你的原始文案内容",
  "targetAudience": "male" // 或 "female"
}
```

**响应**:
```json
{
  "analysis": "爆款分析内容...",
  "similarCopywriting": [
    "生成的文案1",
    "生成的文案2",
    "生成的文案3"
  ]
}
```

## 分析维度

AI 会从以下 7 个维度分析你的文案：

1. **服装类型识别**: 识别具体的服装品类（外套/裤子/裙子等）
2. **爆款公式拆解**: 分析文案使用的结构公式
3. **痛点分析**: 识别文案解决的穿搭痛点
4. **情绪价值分析**: 分析传递的情绪和共鸣点
5. **表达技巧分析**: 修辞手法、口语化表达等
6. **话题标签分析**: hashtag 的选择策略
7. **使用场景识别**: 穿着场景的描述方式

## 文案特色

### 针对女生群体的文案

- **人设**: 25 岁独立自信的女生博主
- **语气**: 年轻、活泼、真诚，像闺蜜聊天
- **核心**: 突出独立自主、勇敢做自己的态度
- **称呼**: "姐妹们"、"宝子们"、"我"
- **情感**: 传递力量感和自信，不只是甜美可爱
- **hashtag**: 偏向独立、自我、态度、女性力量

### 针对男生群体的文案

- **视角**: 25 岁女生博主推荐男装（不暴露感情状态）
- **语气**: 专业、真诚，像懂穿搭的朋友推荐
- **核心**: 强调实用性、性能、版型、显高显瘦
- **表达**: "男生穿这个很帅"、"帮朋友选的"、"男生显高神器"
- **禁忌**: 不使用"男朋友"、"对象"、"男友"等词汇
- **hashtag**: 偏向男装穿搭、男生显高、懒人穿搭

## 最佳实践

### ✅ 推荐做法

1. **真实体验**: 加入具体的穿着感受和使用场景
   - "昨天穿去公司，同事都问哪买的"
   - "实测洗后不变形"

2. **具体细节**: 描述产品的具体特征
   - "麻花针织纹理"
   - "半拉链设计"
   - "高腰A字版型"

3. **痛点解决**: 明确指出解决的具体问题
   - "不会搭的男生"
   - "小个子女生的福音"
   - "梨型身材救星"

4. **场景画面**: 营造具体使用场景
   - "早八通勤穿它气场全开"
   - "过年拜年亲戚都夸好看"

5. **行动号召**: 引导用户采取行动
   - "喜欢的姐妹千万别错过"
   - "男生衣橱必备基础款"

### ❌ 避免做法

1. **过度正式**: "本产品采用先进技术..."
2. **空泛描述**: "这件衣服真的很好看"
3. **标题过长**: 超过 20 字
4. **生硬列举**: "【适合场景】1.通勤 2.约会"
5. **暴露感情**: "帮我男朋友买的"（男装文案）
6. **距离感表达**: "作为模特我..."

## 生成的文案类型

系统会生成 3 种不同风格的文案：

1. **爆款公式完整版**: 遵循经典爆款结构，转化率高
2. **场景自然融入版**: 将场景描述自然融入正文
3. **专业分析版**: 强调专业度和细节分析

每种文案都包含：
- 吸引人的标题
- 自然的场景描述
- 至少 5 个相关 hashtag

## 技术细节

- **AI 模型**: DeepSeek Chat V3
- **Temperature**: 0.8（保证创意性和多样性）
- **Max Tokens**: 2000
- **响应时间**: 通常 3-8 秒

## 故障排除

**问题**: API 返回 400 错误
- **解决**: 检查是否输入了原始文案

**问题**: 生成的文案不符合预期
- **解决**: 尝试调整原始文案，使其更符合爆款公式结构

**问题**: 响应时间过长
- **解决**: 检查网络连接和 OpenRouter API 状态

## 环境变量

确保在 `.env.local` 中配置了以下变量：

```bash
OPENROUTER_API_KEY=your_api_key_here
SITE_URL=http://localhost:3000
SITE_NAME=AI Fashion Generator
```
